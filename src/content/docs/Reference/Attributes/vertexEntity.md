### `vec2 clrwl_vertexEntity;`

**Sent to fragment**: NO  
**Remaps**: `mc_Entity`  

---

The block ID and render type for terrain-like geometries, equivalent to Iris' [`mc_Entity`](https://shaders.properties/current/reference/attributes/mc_entity/).
